# Three.js Visualization Project

Веб-приложение с интерактивной 3D-сценой на Three.js.

## Установка и запуск

```bash
npm install
npm run dev
```

Откройте браузер по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

Результат будет в папке `dist/`

## Part 1

Реализовано:
- базовая 3D-сцена на Three.js;
- управление камерой через OrbitControls;
- 3 источника света:
  - AmbientLight,
  - DirectionalLight,
  - PointLight;
- тени объектов;
- текстура на кубе;
- не менее 5 объектов в сцене;
- минимум 2 объекта созданы через BufferGeometry;
- HTML-элементы управления параметрами сцены:
  - интенсивность направленного света,
  - цвет точечного света,
  - цвет куба.

## Управление

- мышь — вращение камеры;
- колесо мыши — приближение и отдаление;
- панель управления — изменение параметров освещения и цвета куба.
