# Three.js Visualization Project

Веб-приложение с интерактивной 3D-сценой на Three.js.

## Установка и запуск

```bash
npm install
npm run dev
```

Откройте браузер по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

Результат будет в папке `dist/`

## Part 1

Реализовано:
- базовая 3D-сцена на Three.js;
- управление камерой через OrbitControls;
- 3 источника света:
  - AmbientLight,
  - DirectionalLight,
  - PointLight;
- тени объектов;
- текстура на кубе;
- не менее 5 объектов в сцене;
- минимум 2 объекта созданы через BufferGeometry;
- HTML-элементы управления параметрами сцены:
  - интенсивность направленного света,
  - цвет точечного света,
  - цвет куба.

## Part 2

Дополнительно реализовано:
- загрузка пользовательских 3D-моделей;
- поддержка форматов:
  - OBJ,
  - GLTF / GLB,
  - FBX;
- загрузка моделей через выбор файла;
- загрузка моделей через drag-and-drop;
- выбор объекта для взаимодействия через выпадающий список;
- выбор объекта кликом по сцене с помощью raycasting;
- перемещение, вращение и масштабирование выбранного объекта через TransformControls;
- ручное изменение позиции, поворота и масштаба через HTML-поля ввода;
- синхронизация значений в панели управления с текущим состоянием выбранного объекта.

## Управление

- мышь — вращение камеры;
- колесо мыши — приближение и отдаление;
- панель управления — изменение араметров освещения и цвета куба;
- выбор объекта — через список или кликом по сцене;
- трансформация объекта — через поля ввода или TransformControls;
- загрузка модели — через выбор файла или перетаскивание в окно приложения.
